# AGI WAR Knowledge Base

> 최종 업데이트: 2026-04-28

## 서비스 개요

- **서비스명**: AGI WAR (에이지아이 워)
- **URL**: https://ai-war.vercel.app
- **슬로건**: 20개 AI 군단으로 펼치는 전략 카드 배틀
- **설명**: 2030년을 배경으로 한 턴제 카드 전략 게임. 실제 AI 기업(OpenAI, Anthropic, Google 등)을 모티브로 한 38개 AI 팩션의 카드를 수집하고, 스토리 모드와 실시간 PVP를 즐길 수 있습니다.
- **가격**: 무료 플레이 (인게임 재화로 운영)
- **개발사**: 계발자들 (Vibers)

## 주요 기능

### 카드 수집
- **96종 이상의 카드**: Common, Rare, Epic, Legendary, Mythic, Commander 6단계 등급
- **38개 AI 팩션**: ChatGPT, Claude, Gemini, Grok, Midjourney, Stable Diffusion, Suno 등 실제 AI 브랜드 기반
- **카드 생성**: 팩션 구독 후 시간 경과로 자동 생성
- **카드 강화/합성**: 카드를 강화하거나 합성하여 더 강력한 카드 획득

### 스토리 모드
- **시즌 1: GENESIS** (5챕터, 50스테이지)
- 2030년 AI 전쟁을 배경으로 한 시네마틱 스토리
- 캐릭터: Chip(메인 AI), Sam(보안 AI), Dario(개발자), Elon(CEO), Grok(어시스턴트)
- 한국어/영어 대사 지원

### PVP 실시간 대전
- **랜덤 매칭**: 실시간으로 다른 플레이어와 카드 배틀
- **친구 대전**: 코드 공유로 친구와 1:1 대전
- **모드**: Sudden Death, Tactics, Strategy, Double
- Firebase Realtime Database 기반 실시간 동기화

### 카드 클래시
- 카드를 베팅하는 가위바위보 스타일 대전
- 방 코드로 친구와 플레이
- 승리 시 상대 카드 획득

### 미니게임
- 다양한 모드의 간단한 카드 배틀

### 상점
- **Standard Supply** (500 코인): 카드 5장 (Common 60%, Rare 30%, Epic 10%)
- **Premium Supply** (1,000 코인): 카드 5장 (Common 40%, Rare 40%, Epic 15%, Mythic 5%)
- **Token Supply** (1,000 토큰): 카드 1장 (Common 70%, Rare 25%, Epic 5%)
- **데이터 교환**: 100 코인 → 1,000 토큰

### 소셜
- 친구 추가/검색
- 실시간 온라인 상태
- 채팅방

### 기타
- **연구소**: 스탯 연구 트리로 능력 강화
- **일일 보상**: 14일 사이클 (10~150 코인)
- **퀘스트**: 일일/주간/업적 퀘스트 9종
- **랭킹**: 글로벌 리더보드
- **도감**: 전체 카드 백과사전

## 사용 방법

### 1. 회원가입 (30초)
1. ai-war.vercel.app 접속
2. "무료로 시작하기" 클릭
3. 이메일/비밀번호 또는 Google 계정으로 가입
4. 닉네임 설정 → 스타터팩(카드 5장) 지급

### 2. 첫 번째 전투
1. 메인 대시보드 → "작전 지역" (BATTLE FIELD) 클릭
2. 스테이지 1-1 선택
3. 보유 카드 중 5장 선택 → 전투 시작
4. 상성(가위바위보) 시스템으로 승패 결정

### 3. 카드 획득 방법
- **스타터팩**: 가입 시 무료 5장
- **상점 구매**: 인게임 코인/토큰으로 팩 구매
- **카드 생성**: 팩션 구독 → 시간 경과 후 카드 자동 생성
- **전투 보상**: 스토리/PVP 승리 시 코인+경험치 획득
- **일일 보상**: 매일 로그인 시 코인 지급

### 4. PVP 대전 방법
1. 메인 → PVP 아레나 선택
2. 모드 선택 (Sudden Death / Tactics 등)
3. "랜덤 매칭" 또는 "친구 대전" 선택
4. 카드 5~6장 선택 → 실시간 대전

## FAQ

**Q: 게임은 무료인가요?**
A: 네, 완전 무료입니다. 모든 기능을 인게임 재화(코인, 토큰)로 이용할 수 있습니다.

**Q: 어떤 기기에서 플레이할 수 있나요?**
A: 웹 브라우저가 있는 모든 기기(PC, 모바일, 태블릿)에서 플레이 가능합니다. PWA로 설치하면 앱처럼 사용할 수 있습니다.

**Q: 카드는 어떻게 얻나요?**
A: 상점에서 코인/토큰으로 구매, 팩션 구독 후 자동 생성, 스토리/PVP 보상 등 다양한 방법이 있습니다.

**Q: PVP에서 상대가 나가면 어떻게 되나요?**
A: 상대가 접속을 끊으면 15초 후 자동으로 승리 처리됩니다.

**Q: 카드 등급은 어떤 것이 있나요?**
A: Common → Rare → Epic → Legendary → Mythic → Commander 순으로 6단계입니다. Commander는 각 AI 팩션의 CEO를 모티브로 한 최상위 카드입니다.

**Q: 전투 시스템은 어떻게 작동하나요?**
A: 가위바위보 상성 시스템(EFFICIENCY > FUNCTION > CREATIVITY > EFFICIENCY)이 기본이며, 같은 상성일 때는 전투력으로 승패가 결정됩니다.

**Q: 시즌 1 스토리는 몇 챕터까지 있나요?**
A: 총 5챕터, 50개 스테이지입니다. 각 챕터는 10개 스테이지로 구성되며, 5번째와 10번째 스테이지는 보스전입니다.

**Q: 일일 보상은 언제 초기화되나요?**
A: 매일 오전 6시(KST)에 초기화됩니다.

**Q: 버그나 문제를 발견하면 어디에 신고하나요?**
A: 게임 내 "고객지원" 메뉴 또는 설정에서 문의할 수 있습니다.

**Q: 계정을 삭제할 수 있나요?**
A: 설정 메뉴에서 계정 삭제를 요청할 수 있습니다.

## 기술 정보

- **프론트엔드**: Next.js 16 (App Router) + TypeScript + Tailwind CSS
- **백엔드**: Firebase (Realtime Database + Firestore + Auth)
- **배포**: Vercel
- **실시간 통신**: Firebase Realtime Database (PVP, 알림)
- **PWA**: 오프라인 지원, 홈 화면 설치 가능
