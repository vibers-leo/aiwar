## 핵심 목표 (Key Goal)

코워크 환경을 AGI WAR 프로젝트에 연결하고, 코워크에서 수행 가능한 작업 범위를 파악하여 클로드코드와의 역할 분담 체계를 수립한다.

---

## 대화 내용 요약

### 1. 프로젝트 연결 완료
- LEO님이 코워크에 AGI WAR 프로젝트 폴더를 마운트
- CLAUDE.md, OKR.md 등 프로젝트 핵심 문서 접근 확인
- 기존 세션 로그(2025-12 ~ 2026-02) 확인 완료

### 2. 코워크 활용 가능 영역 리스트업

#### 📋 프로젝트 관리
- OKR 주간/월간 진행률 분석 및 리포트 생성
- 세션 로그 백업 및 정리 (코워크 대화 내용 보존)
- 문서 작성 (PPTX, DOCX, PDF, XLSX 직접 생성)
- Google Calendar 연동 일정 관리

#### 📣 마케팅 & 와디즈 펀딩 (OKR Objective 2)
- 와디즈 펀딩 페이지 카피라이팅 및 콘텐츠 제작
- Canva 연동으로 SNS 포스트, 배너, 카드뉴스 디자인
- 마케팅 전략 문서 (티저 캠페인, 콘텐츠 캘린더, 아웃리치 템플릿)
- Gmail 연동으로 이메일 마케팅 드래프트 작성

#### 📊 데이터 & 분석
- KPI 트래킹 스프레드시트 제작
- 경쟁 분석 및 벤치마크 리포트
- 펀딩 시뮬레이션, 리워드 원가 계산

#### 🎨 디자인 & 콘텐츠
- Canva로 게임 소개 자료, 카드 쇼케이스 이미지 제작
- Figma 연동으로 디자인→코드 변환
- 투자자 피칭 덱, 발표자료 PPTX 제작

#### 🔄 자동화
- 정기 리포트/리마인더 스케줄링
- 반복 작업 자동화

### 3. 역할 분담 구조
- **클로드코드 (로컬+클라우드)**: 코딩, 디버깅, 빌드, 테스트, Firebase 작업
- **코워크**: 프로젝트 관리, 마케팅, 문서 작성, 디자인, 분석, 자동화

### 4. 세션 로그 규칙
- 코워크의 모든 대화 내용은 세션 로그에 한글 제목/한글 내용으로 저장
- 저장 경로: `frontend/session-logs/`
- 형식: `YYYY-MM-DD_HHMMSS_한글제목.md`
