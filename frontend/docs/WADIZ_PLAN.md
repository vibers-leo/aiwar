# 와디즈(Wadiz) 펀딩 프로젝트 런칭 계획

## 1. 프로젝트 개요
- **프로젝트 명**: AI WAR - 인류 최후의 지휘관이 되어라 (가제)
- **카테고리**: 게임 / 모바일 / 전략
- **목표 금액**: 500만원 (초기 달성률 100% 목표)
- **컨셉**: AI가 지배하는 세상, 나만의 덱으로 전장을 지배하는 **하이퀄리티 AI 일러스트 카드 배틀**

## 2. 펀딩 상세 페이지 구성 (스토리텔링)

### **Intro: 충격과 몰입**
- **헤드 카피**: "당신의 전략이 AI의 연산을 뛰어넘을 수 있을까?"
- **이미지**: 최고 등급(Mythic) 카드의 화려한 홀로그램 움직임 (GIF)
- **내용**: 게임의 디스토피아적 세계관 소개. 단순한 카드 게임이 아닌, 살아있는 AI와의 전쟁임을 강조.

### **Point 1: 압도적인 비주얼 (AI Art)**
- **내용**: 일반적인 TCG와 차별화된, 생성형 AI로 만들어진 초고해상도 캐릭터 일러스트.
- **자료**: 주요 진영(사이버펑크, 메카닉 등)별 대표 캐릭터 확대 샷.

### **Point 2: 전략의 깊이**
- **내용**: 단순한 스탯 싸움이 아닌, 속성/상성/턴 수를 고려한 치열한 수싸움.
- **자료**: 4가지 승부 모드(단판, 2장, 전술, 전략) 플레이 움짤.

### **Point 3: 멈추지 않는 진화**
- **내용**: 내 카드를 합성하고 강화하여 나만의 '유니크' 등급 카드를 만드는 시스템.

## 3. 리워드 설계 (Backup Plan)

| 리워드명 | 가격 | 구성품 | 비고 |
| :--- | :--- | :--- | :--- |

| **커맨더 패키지** | 49,000원 | 3,000 다이아 + **펀딩 한정판 테두리** + 전설 카드 확정권 | 주력 상품 |
| **오버로드 에디션** | 99,000원 | 10,000 다이아 + **유니크(나만의) 카드 제작권** + 명예의 전당 이름 등재 | 고액 후원자용 |

## 4. 제작 필요한 홍보 콘텐츠 리스트
1.  **메인 배너 이미지**: 와디즈 썸네일용 (강렬한 타이포그래피 + 대표 캐릭터)
2.  **게임 플레이 GIF**: 실제 PVP, 가챠 연출, 카드 강화 장면 녹화.
3.  **카드 쇼케이스**: 등급별(일반~신화) 카드 디자인 나열 이미지.
4.  **세계관 영상/이미지**: 게임 접속 시 나오는 인트로 화면 활용.

## 5. 실행 계획
1.  `/app/showcase` 페이지를 **"와디즈 펀딩 프리뷰"** 페이지로 리뉴얼.
2.  `generate_image` 툴을 사용하여 부족한 고해상도 홍보용 이미지 생성.
3.  실제 게임 플레이 화면을 캡처하여 상세 페이지에 삽입할 GIF 제작 (브라우저 녹화 활용).
