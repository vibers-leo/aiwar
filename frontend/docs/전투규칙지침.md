## 1. ⚡ 단판 승부 (Sudden Death)

### 규칙
1. 5장의 카드를 선택합니다.
2. 1~5라운드 출전 순서를 배치합니다.
3. **첫 번째 라운드에서 이기는 쪽이 즉시 승리합니다.**
4. 무승부 시 다음 라운드로 진행합니다.

### 특징
- 가장 빠른 전투 방식
- 심리전: 첫 라운드에 무엇을 배치할지가 핵심
- 주로 보스전에서 사용

### 승리 조건
```
1선승제: 먼저 1승을 달성하면 즉시 승리
```

---

## 2. 🎯 전술 승부 (Tactical Duel)

### 규칙
1. 5장의 카드를 선택합니다.
2. 1~5라운드 출전 순서를 배치합니다.
3. 순서대로 1:1 대결을 진행합니다.
4. **먼저 3승을 달성하는 쪽이 승리합니다.**

### 특징
- 스토리 모드의 기본 전투 방식
- 상대 덱을 보고 상성을 읽어 배치하는 것이 핵심
- 최대 5라운드까지 진행 가능

### 승리 조건
```
3선승제: 먼저 3승을 달성하면 승리 (3-0, 3-1, 3-2 가능)
```

---

## 3. 🎴 두장 승부 (Two-Card Battle)

### 규칙
1. **6장**의 카드를 선택합니다.
2. 각 라운드마다 2장씩 출전합니다. (총 3라운드)
3. **⏱️ 5초 선택**: 2장 중 **5초 안에 1장을 선택**하여 최종 승부!
4. 라운드별 선택된 1장의 카드로 승부합니다.
5. **먼저 2승을 달성하는 쪽이 승리합니다.**

### 특징
- **순발력 게임**: 5초 안에 판단하고 선택해야 함
- 심리전 + 반응속도의 조합
- 강한 카드 2장을 한 라운드에 몰아넣을지 분산할지 전략적 선택
- 챕터 2 초반에 등장

### 라운드 흐름
```
[라운드 시작] → [2장 카드 공개] → [⏱️ 5초 타이머] → [1장 선택] → [승부 판정]
```

### 승리 조건
```
2선승제: 먼저 2승을 달성하면 승리 (2-0, 2-1 가능)
```

---

## 4. 🌪️ 전략 승부 (Strategy Battle)

### 규칙
1. **6장**의 카드를 선택합니다. (5장 + 1장 히든카드)
2. 1~5라운드 출전 순서를 배치합니다.
3. **3라운드가 특수 라운드입니다:**
   - 3-1 라운드: 기본 카드 대결
   - 3-2 라운드: 히든 카드 대결
   - 3-1, 3-2 **둘 다 승리**: 승점 **2점**
   - 3-1 패배, 3-2 승리: 3-2 승자가 승점 **1점**
   - 3-1 승리, 3-2 패배: 3-1 승자가 승점 **1점**
4. **먼저 승점 3점을 획득하면 승리합니다.**

### 특징
- 가장 복잡하고 전략적인 전투 방식
- 히든 카드 배치가 승패를 결정
- 3라운드에서 2점을 획득할 수 있으므로 역전 가능
- 챕터 4~5에서 사용

### 승점 획득 방법
| 라운드 | 승리 시 획득 승점 |
|--------|----------------|
| 1, 2, 4, 5 라운드 | 1점 |
| 3라운드 (3-1 + 3-2 모두 승리) | **2점** |
| 3라운드 (한 쪽만 승리) | 1점 (해당 라운드 승자에게) |

### 승리 조건
```
승점 3점 선취: 먼저 승점 3점을 획득하면 승리
```

## 📊 상성 시스템 (모든 모드 공통)

```
효율(EFFICIENCY/Rock) > 기능(FUNCTION/Scissors)
기능(FUNCTION/Scissors) > 창의(CREATIVITY/Paper)
창의(CREATIVITY/Paper) > 효율(EFFICIENCY/Rock)
```

### 판정 우선순위
1. **상성** (가위바위보): 우위 타입이 무조건 승리
2. **세부 스탯**: 같은 타입일 경우 해당 타입 스탯 비교
3. **총 전투력**: 세부 스탯도 동일하면 totalPower 비교
4. **무승부**: 모두 동일하면 무승부
