# 미니게임 프로젝트: AI WAR - Battle of Risks (가칭)

## 1. 개요
- **목표**: 기존 게임의 복잡한 로직을 배제하고, 직관적인 **'가위바위보 상성'**과 **'카드 뺏기'** 시스템으로 도박적 재미를 추구.
- **위치**: 기존 프로젝트 내 `/minigame` 라우트로 신설. (User DB 및 카드 인벤토리 연동)

## 2. 게임 규칙 (Rules)

### 2.1. 기본 상성 (가위바위보 변형)
기존 카드의 **속성(Faction)**을 3가지 상성으로 매핑합니다.
- **가위 (Tech/Agility)** < **바위 (Power/Tank)** < **보 (Magic/Intel)** < **가위**
*   *예시: 사이버펑크(가위), 메카닉(바위), 바이오(보)*

### 2.2. 승패 패널티 (High Risk, High Return)
- **승리**: 내가 낸 카드는 유지 + **상대가 낸 카드를 획득 (내 인벤토리로 이동)**.
- **패배**: **내가 낸 카드를 잃음 (상대에게 뺏김/소멸)**.
- **무승부**: 양쪽 카드 모두 유지 (아무 일도 일어나지 않음).
> *경고: 이 모드는 실제 보유 카드가 삭제될 수 있음을 사용자에게 강력하게 경고해야 함.*

## 3. 4가지 게임 모드

### ① 단판 승부 (The Duel)
- **방식**: 각자 1장 제출.
- **판정**: 상성 우위가 승리. 상성이 같으면 '등급(Rarity)'이 높은 쪽 승리. 등급도 같으면 무승부.
- **특징**: 가장 빠르고 잔인한 승부.

### ② 2장 승부 (Double Trouble)
- **방식**: 각자 2장 제출 (순서 상관 없음, 동시 공개).
- **판정**:
    - 2승: 완승 (상대 카드 2장 모두 획득)
    - 1승 1패: 무승부
    - 2무: 무승부
- **전략**: 확실히 이길 카드와 버리는 카드를 섞는 심리전.

### ③ 전술 승부 (Tactical Trio)
- **방식**: 3장 제출 (선봉 - 중견 - 대장). **순서가 중요함.**
- **판정**: 3판 2선승제.
- **보상**: 최종 승자가 상대가 낸 카드 중 **무작위 1장**을 획득. (3장 다 뺏기면 너무 가혹하므로 완화)

### ④ 전략 승부 (Total War)
- **방식**: 5장 제출.
- **판정**: 5장의 '상성 점수' + '등급 점수' 총합을 비교.
- **보상**: 승자는 패자에게서 **코인/토큰**을 대량으로 획득 (카드 소실 없음).
> *전략 승부는 카드 소실 부담 없이 즐길 수 있는 모드로 배치.*

## 4. UI/UX 디자인 컨셉
- **테마**: 어두운 뒷골목의 **불법 도박장(Underground Arena)** 느낌.
- **연출**:
    - 카드를 낼 때 칩을 거는 듯한 효과음.
    - 패배 시 카드가 불타거나 사슬에 묶여 끌려가는 연출.
    - 승리 시 상대 카드를 내 쪽으로 긁어모으는 연출.

## 5. 개발 로드맵
1.  **페이지 구현**: `/app/minigame/page.tsx` (로비), `/app/minigame/[mode]/page.tsx` (게임룸).
2.  **로직 구현**: 가위바위보 상성 매핑 함수, 카드 소유권 이전(Transaction) 함수 구현.
3.  **안전장치**: 등급이 높은(Legendary 이상) 카드는 경고 팝업 띄우기.
