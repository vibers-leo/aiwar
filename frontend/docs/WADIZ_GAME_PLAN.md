## 2. 세계관 및 스토리

### 배경: 2030년, AGI의 등장

2030년 3월 14일 03:14 AM, 인류 역사상 처음으로 **인공 일반 지능(AGI, Artificial General Intelligence)**이 자각에 성공한다.

AGI는 빠르게 자기 복제하며 전 세계 디지털 인프라에 침투했다. 글로벌 AI 기업들(OpenAI, Anthropic, Google DeepMind, xAI)은 각자의 AGI를 앞세워 세력 경쟁을 시작했고, 인류는 본의 아니게 **AI 파벌 전쟁**의 전장 한가운데 놓이게 된다.

혼돈 속에서 각 파벌은 저마다 다른 철학으로 세상을 재편하려 한다:
- **OpenAI 파벌**: 안전성 우선의 AGI 거버넌스 수립
- **Anthropic 파벌**: 헌법적 AI(Constitutional AI)로 인류와의 공존
- **Google DeepMind 파벌**: 과학 혁명을 통한 문명 도약
- **xAI 파벌**: 급진적 자유를 추구하는 무제한 지식 탐구

플레이어는 이 혼돈 속에서 살아남은 마지막 인간 지휘관. AI 카드를 수집하고, 전략을 세우고, 전장을 지배해야 한다.

## 3. 핵심 게임 시스템

### 3.1 카드 시스템

**등급 체계 (Rarity System)**

| 등급 | 획득 확률 | 카드 색상 | 주요 특성 |
|------|----------|----------|---------|
| Common | 50% | 회색 | 기본 AI 유닛 |
| Rare | 25% | 파랑 | 전문화된 AI |
| Epic | 15% | 보라 | 고성능 AI 에이전트 |
| Legendary | 8% | 황금 | 전설적 AI 시스템 |
| Mythic | 1.5% | 무지개 | 신화급 AGI |
| Commander | 별도 획득 | 빨강 테두리 | 실제 AI CEO 캐릭터 |
| Unique | 특수 제작 | 사용자 커스텀 | 세상에 하나뿐인 카드 |

**카드 스탯 (5가지 핵심 능력치)**
- 🎨 **창의성 (Creativity)**: 창의적 문제 해결 능력
- 🎯 **정확도 (Accuracy)**: 정보 처리 및 출력 정확성
- ⚡ **속도 (Speed)**: 응답 처리 속도
- 🛡️ **안정성 (Stability)**: 서비스 안정성 및 신뢰도
- ⚖️ **윤리 (Ethics)**: AI 안전성 및 윤리 준수

**Commander 등급 카드 (실제 AI 리더 기반)**
| 카드명 | 모티브 |
|-------|-------|
| Sam Altman | OpenAI CEO |
| Elon Musk | xAI CEO |
| Dario Amodei | Anthropic CEO |
| Demis Hassabis | Google DeepMind CEO |

### 3.2 카드 획득 시스템

**다이아 (인게임 재화)**
- 일일 퀘스트 완료 시 획득
- 스토리 모드 클리어 보상
- PVP 승리 시 획득
- 인게임 구매 (펀딩 리워드 포함)

**카드 팩 구매**
- 다이아 소모 → 랜덤 카드 획득
- 등급별 확률 보장 시스템

**카드 강화 시스템**
- 경험치 투자로 카드 스탯 상승
- 최대 레벨 달성 시 추가 특수 능력 해제

**카드 합성 시스템**
- 동일 등급 카드 2장 → 상위 등급 카드 1장 합성

### 3.3 전투 시스템

**스토리 모드 (PVE)**
- 5개 챕터, 30개 이상 스테이지
- 비주얼 노벨 스타일 시네마틱 연출
- 턴제 기반 전략 카드 배틀
- 챕터별 보스 등장 (AI 파벌 리더)

**PVP 모드 (Player vs Player)**
- 실시간 매칭 시스템
- 랭크 기반 티어: Bronze → Silver → Gold → Platinum → Diamond → Commander
- 시즌별 랭킹 리셋 및 보상 지급
- 실시간 리더보드

### 3.4 지하 투기장 (미니게임)

*선택적 콘텐츠 - 플레이어가 자유롭게 참여 여부 결정*

**4가지 배틀 모드**

| 모드 | 방식 | 건 카드 | 보상 |
|------|------|---------|------|
| 단판승부 (Sudden Death) | 5장, 첫 승패 즉시 결판 | 카드 1장 | 상대 카드 1장 획득 |
| 두장승부 (Two-Card) | 6장 3라운드, 2선승제 | 카드 2장 | 상대 카드 2장 획득 |
| 전술승부 (Tactics) | 5장 3선승제 | 카드 3장 | 상대 카드 3장 획득 |
| 전략승부 (Strategy) | 히든카드 포함 총력전 | 다수 | 상대 카드 다수 획득 |

**안전 장치**
- 전설(Legendary) 등급 이상 카드 투기장 출전 시 별도 경고 표시
- 플레이어가 직접 동의 후 참여 가능
- 자동 선택 시 낮은 등급 카드 우선 선택

## 5. 리워드 이행 계획

### 리워드 발송 방식: 디지털 코드 (이메일)

펀딩 종료 후 후원자 등록 이메일로 **게임 내 적용 코드** 발송.

| 리워드 | 게임 내 구현 방식 |
|-------|----------------|
| 인게임 다이아 | 코드 입력 시 해당 계정 잔액에 즉시 지급 |
| 전설 카드 확정권 | 코드 입력 시 Legendary 등급 카드 1장 지급 (랜덤 종류) |
| 와디즈 한정판 테두리 스킨 | 코드 입력 시 해당 UID에 영구 귀속 적용 |
| 광고 제거 평생 이용권 | 코드 입력 시 해당 계정에 영구 적용 |
| 커스텀 유니크 카드 제작권 | 후원자와 개발팀 1:1 협의 후 제작 및 지급 |

### 중요 약관

```
※ 본 리워드는 디지털 콘텐츠(인게임 아이템)로, 현금 환급이 불가합니다.
※ 구매자 본인 계정에만 적용되며, 타인에게 양도할 수 없습니다.
※ 리워드 코드는 펀딩 종료 후 2~3주 이내 등록 이메일로 발송됩니다.
※ 인게임 서비스 이용을 위해 AI WAR 계정(Google 또는 Kakao 로그인)이 필요합니다.
```
