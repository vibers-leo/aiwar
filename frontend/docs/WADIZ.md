## 📄 필요 서류 체크리스트

### 1. 메이커 유형 증빙 서류
개인 (사업자 미등록)이므로:
- [ ] **신분증** (주민등록증 또는 여권)

> ⚠️ 사업자가 있다면 사업자등록증 제출. 현재는 신분증만 필요.

---

### 2. 제작 유형별 서류 (자체 제작 + 무형 서비스)
무형 서비스 자체 제작의 경우, 아래 중 해당하는 서류:
- [ ] **게임 기획서** ← 우리는 이걸로 제출 (무형 서비스는 '기획서'로 대체 가능)

> 기획서 포함 내용:
> - 게임 개요 및 세계관 (AI WAR 2030 AGI 설정)
> - 카드 시스템 설명
> - 리워드 이행 방법 (디지털 코드 발송 방식)
> - 개발 현황 (시즌1 완성, 현재 플레이 가능)

---

### 3. 리워드 품목별 서류 ⭐ 핵심

#### 우리 리워드 분류
| 리워드명 | 와디즈 분류 | 필요 서류 |
|---------|-----------|---------|
| 인게임 다이아 1,000 / 4,000 / 10,000 | **사용처가 명시된 금액형 상품권 > 이용권** | 국세 완납 증명서 |
| 전설 카드 확정권 | **사용처가 명시된 금액형 상품권 > 교환권** | 국세 완납 증명서 |
| 커스텀 유니크 카드 제작권 | **사용처가 명시된 금액형 상품권 > 교환권** | 국세 완납 증명서 |
| 와디즈 한정 카드 테두리 스킨 | **사용처가 명시된 금액형 상품권 > 이용권** | 국세 완납 증명서 |
| 닉네임 배지 | 증정 리워드 (부가 구성품) | **서류 불필요** |
| 광고 제거 평생 이용권 | **사용처가 명시된 금액형 상품권 > 이용권** | 국세 완납 증명서 |

#### 공통으로 필요한 서류
- [ ] **국세 완납 증명서** (국세청 홈택스에서 무료 발급 가능)

#### 추가 조건 (목표금액 기준)
- 결제 예정금액 **3,000만원 미만** → 국세 완납 증명서만 필요 ✅
- 결제 예정금액 3,000만원 이상 → 보증보험 서류 추가 필요 (우리 목표는 500만원이므로 해당 없음)

> ⚠️ 중요: "사용처가 명시된 금액형 상품권"은 **타인에 양도 불가** 조건 필수.
> 리워드 설명에 "양도 불가, 구매자 계정에만 적용" 문구 반드시 포함할 것.

## 📋 와디즈 제출 프로세스

```
1단계: 프로젝트 작성 (메이커 스튜디오)
   - 프로젝트 유형: 펀딩
   - 카테고리: 무형 서비스
   - 리워드 설정 (3종)
   - 서류 첨부 (신분증, 기획서, 국세완납증명서)

2단계: 심사 제출
   - 심사 기간: 영업일 기준 3~7일

3단계: 심사 통과 후 펀딩 오픈
   - 오픈 전 프로젝트 URL 공유 및 사전 홍보 시작

4단계: 펀딩 진행 (30일)

5단계: 성공 종료 후 리워드 발송
   - 디지털 코드 이메일 발송 (2~3주 이내)
```

## 📌 리워드별 양도 불가 설정 방법

와디즈 리워드 등록 시 **"현금 환급 불가 / 타인 양도 불가"** 옵션 체크:
- 다이아 이용권 → 해당 계정에만 지급
- 카드 교환권 → 발급된 코드는 1회만 사용 가능, 계정 귀속
- 테두리 스킨 → 구매자 UID에 영구 귀속
