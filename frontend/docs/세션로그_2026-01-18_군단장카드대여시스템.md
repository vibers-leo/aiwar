# 세션 로그: 군단장 카드 대여 시스템 개선

**날짜**: 2026-01-18  
**작업 시간**: 약 30분  
**주요 목표**: AI 군단 페이지 레이아웃 개선 및 군단장 카드 대여 시스템 정립

---

## 📋 작업 개요

사용자가 AI 군단 구독 시 군단장 카드가 영구 소유되는 문제를 수정하여, 생성 슬롯에 배치된 군단에서만 임시 대여되도록 시스템을 개선했습니다.

---

## ✅ 완료된 작업

### 1. AI 군단 페이지 레이아웃 개선

**파일**: `/frontend/app/factions/page.tsx`

**변경 내용**:
- 티어별 혜택 정보를 왼쪽 사이드바 팁으로 이동
- 상단에 3열 대시보드 추가:
  - 일간 구독 비용 + 현재 보유 코인
  - 토큰 부스트 현황 + 실제 토큰 수
  - 활성 구독 목록
- UserContext 연동으로 실시간 데이터 반영
- 군단 그리드 가시성 개선

**커밋**: `🎨 AI 군단 페이지 레이아웃 개선`

---

### 2. 군단장 카드 대여 규칙 변경

**변경 전 (잘못된 동작)**:
```
구독만 하면 → 인벤토리에 군단장 카드 추가
구독 중인 6개 군단 → 6개 군단장 카드 영구 소유
```

**변경 후 (정확한 동작)**:
```
슬롯에 배치해야만 → 군단장 카드 임시 대여
슬롯에서 제거하면 → 카드 즉시 회수
최대 5장만 소유 가능 (슬롯 5개)
```

**수정 파일**:
- `/frontend/context/UserContext.tsx`
  - `getGenerationSlots()` 기반 대여 로직으로 변경
  - `refreshInventory()` 함수 추가
  - `isRentalCard` 플래그 도입
  - `isLocked=true`로 합성 방지

- `/frontend/lib/inventory-system.ts`
  - `InventoryCard` 인터페이스에 `isRentalCard` 속성 추가

- `/frontend/app/generation/page.tsx`
  - 슬롯 변경 시 `refreshInventory()` 호출

**커밋**: `🎖️ 군단장 카드 대여 시스템 개선`

---

### 3. 군단 배치 모달 개선

**파일**: `/frontend/app/generation/page.tsx`

**추가된 UI 요소**:
- 군단장 카드 미리보기 (전투에서 사용 가능 표시)
- 토큰 부스트 효과 상세 정보:
  - 군단별 특성 (충전량/속도/용량 증가)
  - Before → After 비교 표시
  - 증가분 하이라이트
- 슬롯 제거 시 카드 회수 경고

**모달 예시**:
```
┌────────────────────────────────────┐
│ ⚡ 토큰 부스트 효과                  │
│                                    │
│ Gemini 특성: 🔋 최대 용량 증가      │
│                                    │
│  충전량    │ 충전 주기  │ 최대 용량   │
│ 100 → 100 │ 60분→60분 │ 1000→2000  │
│           │           │ +1000      │
└────────────────────────────────────┘
```

**커밋**: `✨ 군단 배치 모달 개선 - 토큰 부스트 효과 표시`

---

### 4. 기존 군단장 카드 자동 회수 마이그레이션

**파일**: `/frontend/context/UserContext.tsx`

**로직**:
1. 로그인 시 인벤토리에서 군단장 타입 카드 검색
2. 각 카드의 `aiFactionId` 확인
3. 해당 팩션이 슬롯에 없으면 stale로 마킹
4. Firebase에서 삭제 후 로컬 상태 업데이트

**콘솔 로그 예시**:
```
[Migration] Cleaning up 4 stale commander cards...
[Migration] ✅ Removed stale commander: Gemini Commander
[Migration] ✅ Removed stale commander: ChatGPT Commander
```

**커밋**: `🧹 기존 군단장 카드 마이그레이션 - 규칙 위반 카드 자동 회수`
